envuidgid プログラム

envuidgid は指定されたアカウントの uid と gid を示す環境変数を設定して別のプログラムを起動させます。

使い方

     envuidgid account child
account は一つの引数です。 child は一つ以上の引数から成ります。

envuidgid は account の uid を $UID に、account の gid を $GID に設定します。 それから child を起動させます。

envuidgid は UNIX アカウント account を見つけられない、あるいは環境変数用のメモリが足らない、あるいは child を起動できないの場合に、コード 111 で終了します。 そうでなければ、その終了コードは child の終了コードと同じです。
