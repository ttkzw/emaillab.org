daemontools のロギング・ユーティリティについては log mailing list で議論されています。

それは何か？

daemontools は UNIX のサービスを管理するツール集です。

supervise はサービスを監視します。 サービスを起動させ、そのサービスが死んだら再起動させます。 新しいサービスの設定は簡単です: サービスを動かすスクリプト run のあるディレクトリさえあれば、supervise はサービスを起動できます。

multilog はエラーメッセージを一つ以上のログに保存します。 各行にタイムスタンプを付けたり、各ログに特定のパターンに一致した行を含めたり除いたりすることもできます。 ディスクスペースの使用量を制限するために自動的にログを循環させます。 ディスクがいっぱいになったら、一旦停止して再び書き込もうとし、データを失うことはありません。
