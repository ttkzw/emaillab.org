argv0 プログラム

argv0 は指定した 0 番目の引数でプログラムを起動します。

使い方

     argv0 realname zero args
realname は一つの引数です。 zero は一つの引数です。 args は任意の数の引数です。

argv0 はディスク上に realname という名前で格納されているプログラムを起動します。このとき、zero を（realname 以外の） 0 番目の引数に、args を残りの引数に設定します。

適応

プログラムの中には 0 番目の引数に特別な注意を払うものもあります。 argv0 はシェルスクリプトからこのようなプログラムを使いやすくします。

例えば、 

     argv0 /bin/csh -csh
は -csh を 0 番目の引数として /bin/csh を起動します。 /bin/csh はログインシェルであると考え、それ相応に振る舞うでしょう。

別の例として、コマンド 

     tcpserver 0 ftp argv0 tcpd ftpd -l -A
は /etc/inetd.conf における 
     ftp stream tcp nowait root tcpd ftpd -l -A
と同じような効果があります。 tcpd プログラムは 0 番目の引数 ftpd と残りの引数 -l -A で起動されます。 tcpd は tcpserver の -x オプションによって置き換えられることができ、また置き換えられるべきであることを気づいてください。
