date@ プログラム

date@ はリモートホストの日時を出力します。

使い方

     date@ host
date@ は host の TCP 13 番ポート (Daytime) に接続し、受け取ったどんなデータも出力します。 CR (\015) を取り除き、表示できない文字を表示できる形式に変換します。

host が指定されなかったら、date@ はローカルホストに接続します。

コンピュータの中には人間が読むことのできる日時データを 13 番ポートに返答するものもあります。 例えば、tcpserver を使って次にように動いているかもしれません。 

     tcpserver -RHl0 0 13 date
