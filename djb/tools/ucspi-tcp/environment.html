TCP 環境変数

$PROTO は文字列 TCP です。

$TCPLOCALIP はローカルホストの IP アドレス（ドット付き10進表記）です。

$TCPLOCALPORT はローカル TCP ポート番号（10進表記）です。

$TCPLOCALHOST は DNS に載っているローカルホストの名前です。 名前が有効でなければ $TCPLOCALHOST は設定されません。 $TCPLOCALHOST は任意の文字を含めることに注意してください。

$TCPREMOTEIP はリモートホストのIP アドレス（ドット付き10進表記）です。

$TCPREMOTEPORT はリモート TCP ポート番号（10進表記）です。

$TCPREMOTEHOST は DNS に載っているリモートホストの名前です。 名前が有効でなければ $TCPREMOTEHOST は設定されません。 $TCPREMOTEHOST は任意の文字を含めることに注意してください。

$TCPREMOTEINFO は 931/1413/IDENT/TAP プロトコル経由でリモートホストによって与えられる接続に特有の文字列です。 情報が有効でなければ、$TCPREMOTEINFO は設定されません。 $TCPREMOTEINFO は任意の文字を含めることに注意してください。
