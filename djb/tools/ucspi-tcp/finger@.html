finger@ プログラム

finger@ はリモートホストからユーザ情報を取得します。

使い方

     finger@ host user
finger@ は host の TCP 79 番ポート (Finger) に接続し、host へ（CR を付加した）user を送り、受け取ったどんなデータも出力します。 CR を取り除き、表示できない文字を表示できる形式に変換します。

コンピュータの中には user についての情報を 79 番ポートに返答するものもあります。

user が指定されなかったら、finger@ は host へ空白行を送ります。 コンピュータの中にはログインしている全てのユーザについての情報を返答するものもあります。

host が指定されなかったら、finger@ はローカルホストに接続します。
