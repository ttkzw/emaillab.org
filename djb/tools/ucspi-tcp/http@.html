http@ プログラム

http@ はリモートホストからウェブページを取得します。

使い方

     http@ host page port
http@ は host の port 番のポートに接続し、host に 
     GET /page HTTP/1.0
     Host: host
を送り、返答の内容を各行末から CR を取り除いて出力します。

port が指定されなかったら、http@ は 80 番ポート (HTTP) に接続します。

page が指定されなかったら、http@ は host に / を要求します。

host が指定されなかったら、http@ はローカルホストに接続します。
