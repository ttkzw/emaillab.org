ucspi-tcp のインストール

他のソフトウェアと情報一般と同様に、ucspi-tcp には保証書は付いていません。

必要なシステム

ucspi-tcp は UNIX でしか動きません。

インストール

ucspi-tcp パッケージをダウンロードしてください。 ucspi-tcp パッケージの最新版は ucspi-tcp-0.88.tar.gz です。

ucspi-tcp パッケージを展開してください: 

     gunzip ucspi-tcp-0.88.tar
     tar -xf ucspi-tcp-0.88.tar
     cd ucspi-tcp-0.88

ucspi-tcp プログラムをコンパイルしてください: 

     make

ルートになって、/usr/local に ucspi-tcp プログラムをインストールしてください: 

     make setup check

成功を報告したいときは: 

     ( echo 'First M. Last'; cat `cat SYSDEPS` ) \
     | mail djb-sysdeps@cr.yp.to
First M. Last をあなたの名前で置き換えてください。
