mconnect プログラム

mconnect はリモート SMTP サーバに接続します。

使い方

     mconnect host port
mconnect は host の port 番のポートに接続します。 各行に CR を加えて host にその入力を送ります。 その間に、host から受け取ったどんなものも出力します。

port が指定されなかったら、mconnect は 25 番ポート (SMTP) に接続します。

host が指定されなかったら、mconnect はローカルホストに接続します。
