はじめに

電子メール関連の規格の文書のリンク集としてwww.worldtalk.com の "E-Mail Referrences" があります。しかし、海外のサイトであることと、RFC のリンク先が InterNIC であることも合わせて重いので、同じようなものを作ろうと思いました。その後、Internet Mail Consortium の Internet Mail Standards などを見て色々試行錯誤をして現在に至ります。RFC のリンク先は RFC のミラーをしているサイトとして、Ring Server Project を選びました。

注意事項

  • MIME 周りの分類がまだ不十分です。
  • Internet Drafts をすべて網羅しているわけではありません。
  • Internet Drafts に関してはその文書の更新によりリンクが切れる場合があります。そのときはファイル名の最後の数字に 1 を加えて下さい。恐らく新しい文書が表示されるはずです。（このページは月に一回程度更新を行うつもりです。）
  • X.400, FAX, Voice などのインタネット・メイル以外とのゲイトウェイ及びX.500, LDAPなどのディレクトリシステムとのインターフェイス関連は除いています。
