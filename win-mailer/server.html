試験環境

Last modified: Sun Jun 11 17:04:15 2000

OS

サーバマシンの OS は Linux (Kondara MNU/Linux 1.1) です。

SMTPサーバ

SMTPサーバには qmail を用いました。バージョンは 1.03 です。SMTP-AUTH のために、qmail-smtpd-auth パッチを当て、CRAM-MD5 用の認証プログラムには cmd5checkpw を、平文用の認証プログラムには checkpassword を用いました。

POP3サーバ

POP3サーバには qmail パッケージ付属のものを用いました。APOP 認証パッケージとして qmapop を用いました。バージョンは 0.51 です。

IMAPサーバ

IMAPサーバには UW IMAP server を用いました。バージョンは 4.7c2 です。qmail に対応するためにいくつかの修正を行っています。

LDAPサーバ

LDAPサーバには OpenLDAP を用いています。バージョンは 1.2.10 です。
